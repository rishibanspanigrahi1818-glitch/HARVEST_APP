# 🌱 AgroSense: Crop Intelligence System

## 📌 Problem
Farmers often lack access to real-time insights on crop health and market prices, leading to poor decision-making and financial loss.

## 💡 Solution
AgroSense is an AI-powered system that helps farmers:
- Monitor crop conditions
- Analyze agricultural data
- Get recommendations for the best market (mandi) to sell produce

## ⚙️ Features
- Crop monitoring and analysis
- Market price comparison
- Decision support system for farmers
- Simple and user-friendly interface

## 🧠 Tech Stack
- Python
- Machine Learning
- HTML/CSS (Frontend)
- Data Analysis

## 📊 Future Improvements
- Integration with real-time weather APIs
- Mobile application for farmers
- Live mandi price tracking

## 🚀 Impact
This project aims to improve farmers' income by enabling data-driven agricultural decisions.
